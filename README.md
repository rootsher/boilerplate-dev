## Requirements

* docker `20.10.8` (build `3967b7d`)
* docker-compose `1.29.2` (build `5becea4c`)
* make `3.81`

## Running

### Installation and first running

```shell
$ make setup
```

### Development running

```shell
$ make run
```

### Host availability

After services start - go to [http://project.local](http://project.local)
